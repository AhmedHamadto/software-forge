# Security Policy

## Scope

Software Forge is a skills framework containing prompt-based instructions (Markdown files) and a shell installer. It does not process user data, run servers, or handle credentials.

## Reporting a Vulnerability

If you discover a security issue (e.g., a skill that could cause unintended code execution, or a flaw in the install script), please report it by [opening a GitHub issue](https://github.com/AhmedHamadto/software-forge/issues) with the **bug report** template.

For issues that should not be disclosed publicly (e.g., a skill that could be exploited to exfiltrate data), contact the maintainer directly via GitHub.

## Supported Versions

Only the latest release is supported with security fixes.

| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |
